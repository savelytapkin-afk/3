# Changelog

## v3.4 - 2026-06-25

### 🔧 Исправления
- **Dolphin API endpoint**: обновлён с `POST /v1/browser/start-browser` на актуальный `POST /v1.0/browser_profiles/{id}/start`
- **Нормализация токена**: добавлен `.strip()` при загрузке, сохранении и использовании токена для устранения ошибок из-за пробелов/переносов строк
- **Валидация токена**: добавлен метод `_validate_dolphin_token()` с проверкой перед открытием профилей
- **Обработка ошибок**: явная диагностика 401/403 ошибок авторизации с понятными сообщениями
- **Retry-логика**: добавлены повторные попытки подключения к localhost:3001 (до 3 раз с паузой 3 сек)

### 📝 Документация
- Обновлена версия с v3.3 на v3.4
- Добавлен раздел "Часто встречающиеся ошибки" в README
- Добавлено примечание о требовании платной подписки Dolphin
