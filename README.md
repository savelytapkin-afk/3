# Gmail Sender v3.4

Автоматизированный инструмент для рассылки писем через Gmail с интеграцией CreateAd API и Dolphin Anty.

## 🚀 Функционал

- ✅ Отправка первых писем через Gmail
- ✅ Проверка входящих ответов
- ✅ Автоматическая рассылка персонализированных HTML писем с уникальными ссылками
- ✅ Интеграция с Dolphin Anty для управления профилями
- ✅ Интеграция с CreateAd API для создания объявлений
- ✅ Персонализация писем через переменные {link}, {price}, {name}
- ✅ Профили остаются открыты между этапами
- ✅ Логирование в реал-тайм

## 📋 Требования

- Python 3.8+
- Chrome/Chromium браузер
- Dolphin Anty с **платной подпиской** (для работы API необходима платная подписка)
- API ключи для: Dolphin API, Parser API, CreateAd API

## 💿 Установка

```bash
# Установить зависимости
pip install -r requirements.txt
```

## ⚙️ Настройка

### 1. Создать конфиг

```bash
cp config.json.example config.json
```

Отредактировать `config.json`:
- `token` - JWT токен Dolphin Anty, вставляется как есть без префикса `Bearer`
- `parser_key` - API ключ парсера
- `user_id` - ID в CreateAd
- `api_key` - API ключ CreateAd
- `service_code` - код сервиса (например: vinted_it)

### 2. Создать файл профилей

```bash
cp profiles.txt.example profiles.txt
```

Отредактировать `profiles.txt` - добавить ID профилей Dolphin (один ID на строку).

### 3. Запустить приложение

```bash
python app.py
```

## 🎯 Как использовать

### Этап 1: Отправка первых писем

1. Перейди на вкладку **"🚀 Отправка писем"**
2. Заполни параметры:
   - **Парсер**: API ключ, платформа, страна
   - **CreateAd**: User ID, API Key, Service Code
   - **Управление**: задержка между письмами, макс писем на профиль
3. Нажми **"▶ СТАРТ"**

Процесс:
- Парсинг email-ов с маркетплейса
- Открытие профилей Dolphin
- Отправка первых писем со всеми профилями
- **Профили остаются открыты!**

### Этап 2: Проверка ответов

1. Перейди на вкладку **"💬 Проверка ответов"**
2. Заполни CreateAd параметры (если отличаются от этапа 1)
3. Нажми **"🔍 ПРОВЕРИТЬ ОТВЕТЫ"**

Процесс:
- Проверка входящих писем в каждом открытом профиле
- Для каждого ответа:
   1. Находит email отправителя
   2. Ищет его оригинальные данные в парсере
   3. Создаёт новое объявление в CreateAd
   4. Получает новую уникальную ссылку
   5. Генерирует персонализированный HTML
   6. Отправляет ответ

## 📝 Переменные в письмах

### Первое письмо
- `{price}` - цена товара

### Ответ на письмо
- `{link}` - уникальная ссылка на объявление (от CreateAd)
- `{price}` - цена товара
- `{name}` - имя отправителя (из email)

## 📊 Рекомендуемые значения

- **Задержка между письмами**: 5-10 сек
- **Макс писем на профиль**: 10-20
- **Максимум профилей**: 5-10

## ❓ Часто встречающиеся ошибки

### 🔴 "invalid session token"

**Проблема**: При нажатии "▶ СТАРТ" появляется ошибка `invalid session token`

**Решение**:

1. **Получи новый токен** в Dolphin Anty:
   - Откройте Dolphin Anty
   - Перейдите в **Настройки → API**
   - Скопируйте **API Токен**

2. **Вставь токен** в приложение:
   - Откройте вкладку **"🐬 Dolphin"**
   - Вставьте JWT токен в поле **"API Token"** как есть, без префикса `Bearer`
   - Нажмите **"💾 Сохранить токен"**

3. **Проверь подключение**:
   - Убедитесь, что Dolphin Anty запущена
   - Проверьте, что приложение слушает на `localhost:3001`

4. **Если всё равно не работает**:
   - Перезагрузите Dolphin Anty
   - Очистите `config.json` и создайте новый из `config.json.example`
   - Проверьте брандмауэр — может блокировать `localhost:3001`

### 🔴 "Dolphin Anty недоступен на localhost:3001"

**Проблема**: Приложение не может подключиться к Dolphin Anty

**Решение**:

1. Убедитесь, что Dolphin Anty **запущена**
2. Проверьте, что она слушает на порту **3001**
3. Перезагрузите Dolphin Anty
4. Проверьте брандмауэр/антивирус

## ⚠️ Важно

- Убедись что Gmail аккаунты в профилях залогинены
- API ключи не отправляются никуда (сохраняются только локально)
- Профили открываются только для отправки и проверки
- Закрытие профилей - вручную через Dolphin Anty
- **Для работы с API требуется платная подписка Dolphin Anty**
