# Gmail Sender v3.5

Автоматизация рассылки писем с интеллектуальным парсером.

## 🚀 Функциональность

- ✅ Интеграция с Dolphin Anty для открытия браузер-профилей
- ✅ Динамический парсер для всех популярных маркетплейсов
- ✅ Автоматическая отправка писем через Gmail
- ✅ Проверка входящих писем и автоответы
- ✅ Интеграция с CreateAd API для получения новых ссылок
- ✅ Поддержка Spintax `{вариант1|вариант2}` для персонализации
- ✅ Мультиязычные селекторы Gmail (русский + английский)

## 📋 Требования

- Python 3.8+
- Dolphin Anty (запущен на localhost:3001)
- Google Chrome / Chromium
- Gmail аккаунт

## 🔧 Установка

```bash
pip install -r requirements.txt
```

Для автоматизированной установки можно использовать:

- `install.sh` (Linux/macOS)
- `install.bat` (Windows)

## ▶️ Запуск

```bash
python app.py
```

## 📖 Как использовать

### 1. Настройка Dolphin Anty

- Откройте Dolphin Anty
- Перейдите в Настройки → API
- Скопируйте API токен
- Вставьте в приложении на вкладке "🐬 Dolphin"
- Нажмите "💾 Сохранить токен"

### 2. Загрузка профилей

- Загрузите файл с ID профилей (один ID на строку)
- Максимум 3000 профилей одновременно

### 3. Настройка парсера

- Выберите платформу (Vinted, Depop и т.д.)
- Установите фильтры (цена, категория, новизна и т.д.)
- Нажмите "💾 Сохранить фильтры"
- Протестируйте: "🧪 Тест парсера"

### 4. Создание шаблонов

- **Первое письмо**: используйте `{title}`, `{price}`, `{вариант1|вариант2}`
- **Ответ**: используйте `{link}` (ссылка от CreateAd), `{price}`, `{name}`

### 5. Отправка писем

- Установите CreateAd ключи (User ID, API Key, Service Code)
- Нажмите "▶ СТАРТ"
- Приложение будет отправлять письма с задержкой

### 6. Проверка ответов

- Приложение автоматически:
  1. Находит входящие письма
  2. Ищет отправителя в данных парсера
  3. Запрашивает новую ссылку у CreateAd
  4. Отправляет ответ с новой ссылкой
